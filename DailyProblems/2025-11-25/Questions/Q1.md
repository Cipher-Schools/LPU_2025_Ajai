## **Problem Statement**

Given an array `nums`, create an array `result` where:

`result[i] = product of all elements of nums except nums[i]`.

You **cannot** use division.

### **Example**

Input: `[1, 2, 3, 4]`

Output: `[24, 12, 8, 6]`

---

## **Where Prefix & Suffix Are Used?**

To find the product except at index `i`, you need:

* **Prefix product:** product of all elements **before** `i`
* **Suffix product:** product of all elements **after** `i`

**Final:**

`ans[i] = prefix[i] * suffix[i]`

---

## **Hint**

1. Create a prefix product array
   * `prefix[i] = prefix[i-1] * nums[i-1]`
2. Create a suffix product array
   * `suffix[i] = suffix[i+1] * nums[i+1]`
3. Multiply prefix and suffix values for each index.
