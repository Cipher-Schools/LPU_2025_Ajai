## **Problem Statement**

Given an array `nums`, create a new array where each value is the  **sum of all previous elements including itself** .

### **Example**

Input: `[1, 2, 3, 4]`

Output: `[1, 3, 6, 10]`

---

## **Where Prefix Is Used?**

Running sum  *is literally the prefix sum* :

* `prefix[i] = nums[0] + nums[1] + ... + nums[i]`

---

## **Hint**

1. Maintain a variable `sum`.
2. For each index `i`, do:
   * `sum += nums[i]`
   * store `sum` in the output array.
